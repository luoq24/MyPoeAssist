class PriceDistributionChart(QWidget):
    """价格分布直方图"""
    
    # 依赖: PyQt6.QtCharts 或 matplotlib
    
    def set_data(self, prices: list[float]):
        """设置价格数据"""

class MainWindow(QMainWindow):

        for page in self.pages.values():

        self.page_stack.setCurrentIndex(index)

### 7.2 主题切换机制

```python
class ThemeManager:
    _instance = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance.current_theme = "dark"
        return cls._instance
    
    def apply_theme(self, theme_name: str):
        """应用主题"""
        theme_file = f"styles/theme_{theme_name}.qss"
        with open(theme_file, 'r', encoding='utf-8') as f:
            stylesheet = f.read()
        QApplication.instance().setStyleSheet(stylesheet)
        self.current_theme = theme_name
```

## 八、注意事项

### 8.1 性能优化

| 优化点 | 说明 |
|--------|------|
| 虚拟列表 | 通知列表使用虚拟滚动，避免大量Widget创建 |
| 延迟加载 | 图表数据在页面显示时才加载 |
| 缓存机制 | 缓存已渲染的页面，避免重复初始化 |
| 异步操作 | API请求和数据处理使用异步，避免UI卡顿 |

### 8.2 用户体验

| 体验点 | 说明 |
|--------|------|
| 加载状态 | 所有异步操作显示加载指示器 |
| 空状态 | 无数据时显示友好的空状态提示 |
| 错误处理 | 网络错误、API错误显示明确的错误信息 |
| 操作反馈 | 操作成功/失败显示Toast提示 |
| 快捷键 | 支持常用快捷键（Ctrl+N新建任务等） |

### 8.3 可维护性

| 维护点 | 说明 |
|--------|------|
| 组件解耦 | 组件间通过信号通信，降低耦合 |
| 样式分离 | QSS样式文件独立管理，便于主题切换 |
| 配置外置 | 可配置项提取到配置文件 |
| 日志记录 | 关键操作记录日志，便于排查问题

## 九、总结

本UI实现计划采用"先搭建基础组件，再组装成套"的思路，将UI开发分为7个阶段：

1. **基础框架**：主窗口、导航栏、基础组件库、主题系统
2. **监控任务页**：任务列表、任务编辑、任务操作
3. **通知中心**：通知列表、通知状态、通知操作
4. **词缀统计页**：排行表格、分布图表、组合分析
5. **引路石页**：包装已有功能、词缀库管理、音效设置
6. **设置页**：登录管理、联赛设置、通用设置
7. **集成优化**：功能联调、性能优化、边界处理

通过这种分层实现的方式，可以：
- 降低开发复杂度
- 提高代码复用性
- 便于并行开发
- 方便测试和维护
