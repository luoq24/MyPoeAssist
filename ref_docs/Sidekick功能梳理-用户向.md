# Sidekick — POE2 玩家功能手册

> Sidekick 是一款《流放之路 2》（POE2）游戏辅助工具，运行在游戏上层，通过快捷键在游戏内呼出悬浮窗，提供查价、查资料、资产管理等一系列功能。

---

## 一、核心操作方式

| 操作 | 默认快捷键 | 说明 |
|------|-----------|------|
| 查价 | `Ctrl+D` | 游戏中将鼠标悬停在物品上，按 Ctrl+D 即可呼出查价悬浮窗 |
| 关闭悬浮窗 | `Space` | 关闭当前悬浮窗 |
| 打开 Wiki | `Alt+W` | 在浏览器中打开物品的 Wiki 页面 |
| 快速查找 | `Ctrl+F` | 在游戏内快速搜索同名物品 |
| 聊天宏命令 | `F5` / `F4` / `F9` 等 | 一键发送聊天命令 |

所有快捷键均可在设置中自定义修改。

---

## 二、物品查价（核心功能）

游戏中鼠标悬停物品 → 按 `Ctrl+D` → 弹出查价悬浮窗，包含以下信息：

### 1. 物品详情展示
- 显示物品名称、类型、稀有度、词缀、插槽、要求等完整信息
- 支持 POE2 的独特词缀、符文插槽、技能宝石插槽等展示

### 2. 市集查价（Trade）
- 自动解析物品词缀并生成对应的搜索过滤条件
- 可手动调整过滤条件（词缀数值范围、物品类型、孔色等）
- 点击"Search"按钮在官方市集网站搜索同类型物品
- 列出当前在售的同类型物品及其标价
- **支持 POE2 国服市集（使用英语市集网站）**

### 3. 流通数据（Poe2Scout.com）— POE2 专属
- 显示物品在 POE2 市集中的**实时和歷史成交数据**
- 展示以崇高石、混沌石、神圣石计价的**价格趋势图表**
- 显示**成交量/挂单量**，帮助判断价格可信度
- 数据来源：poe2scout.com

### 4. 通货汇率（poe.ninja）
- 显示物品的**混沌石、崇高石、神圣石等价估值**
- 展示通货**兑换汇率**（如：1 崇高石 = X 混沌石）
- 显示**交易量**和**价格趋势走势图**

### 5. AI 估价（poeprices.info）
- 由算法根据物品词缀自动估算价格
- 显示**估价区间**（最低价 ~ 最高价）和**置信度**
- 可在设置中开启/关闭

### 6. 通货兑换（Exchange）
- 直接查看物品与常见通货的**兑换比率**
- 快速跳转到官方市集的兑换页面

### 7. Wiki 查询
- 悬浮窗内直接点击打开物品的 Wiki 页面
- 可选：**poewiki.net** 或 **poedb.tw**

### 8. Craft of Exile
- 将物品导入 Craft of Exile 在线配装模拟器
- 支持 POE2 模式

## 三、区域危险检测（Area Info）

查看地图/区域物品时，根据用户自定义的正则表达式规则，自动检测地图词缀中是否有**高危词缀**（如元素反射、无法恢复生命等）。

- 匹配到危险词缀 → 红色警告图标 + 列出具体危险词缀
- 无危险词缀 → 绿色安全标识

---

## 四、聊天宏命令（Chat Commands）

在游戏中一键发送预设的聊天命令，支持自定义：

| 预设命令 | 快捷键 | 功能 |
|---------|--------|------|
| `/hideout` | `F5` | 返回藏身处 |
| `/leave` | `F4` | 退出当前队伍 |
| `/exit` | `F9` | 退出游戏到角色选择 |
| `@last ` | `Ctrl+Enter` | 回复上一个私聊你的玩家 |

- `@last` 通配符：自动替换为上一个私聊你的角色名
- 支持自定义快捷键和命令内容
- 支持命令输入后是否按回车发送

---

## 五、财富追踪（Wealth Tracker）

> POE2 暂不支持此功能（代码中已标记 Not Supported for POE2），以下为 POE1 功能留作参考。

通过 GGG 账号 API 自动扫描所有仓库页，追踪总资产变化：

- 按仓库页展示所有物品
- 按物品价值排序
- 展示**资产随时间变化的折线图**
- 展示**各仓库页价值分布图**
- 可设置最低统计价值阈值

---

## 六、工具类功能

### 随手色计算器（Chromatic Calculator）
- 根据装备的力量/敏捷/智力需求，计算洗出指定孔色组合的最优方案
- 显示每种方案的**期望成本（工匠石数量）**和**成功率百分比**

---

## 七、快捷键与操作优化

| 功能 | 默认 | 说明 |
|------|------|------|
| 滚轮切换仓库页 | 开启 | 在仓库上方滚动滚轮切换仓库页 |
| 反向滚轮 | 可选 | 切换滚轮方向 |
| Esc 关闭悬浮窗 | 开启 | 按 Esc 关闭悬浮窗 |
| 鼠标点击外部关闭 | 可选 | 点击悬浮窗外区域自动关闭 |
| 保留剪贴板 | 可选 | 执行操作后恢复剪贴板内容 |
| 保存窗口位置 | 可选 | 记住上次悬浮窗位置 |

---

## 八、多语言与游戏支持

- **支持 POE2 国际服主流语言**：英语、法语、德语、西班牙语、日语、韩语、葡萄牙语、俄语、泰语、中文
- **界面语言**：英语、法语、韩语
- **物品解析语言**：自动匹配游戏内语言
- **联盟选择**：自动获取并支持当前赛季联盟

---

## 九、系统功能

- **自动更新检测**：启动时检查 GitHub 新版本
- **开机静默启动**：可选是否开机自启且静默运行
- **硬件加速**：可选开启 WebView2 硬件加速
- **Docker 部署**：支持以 Web 模式运行（非游戏场景可用）
- **API 状态监控**：实时显示各数据源 API 调用限额和使用状态

---

## 十、数据来源一览

| 数据源 | 用途 | 适用范围 |
|--------|------|---------|
| **Path of Exile 官方市集 API** | 物品搜索、成交结果 | POE1 + POE2 |
| **GGG 账号 API** | 仓库页数据读取（财富追踪） | 仅 POE1 |
| **poe2scout.com** | POE2 物品成交历史与价格趋势 | **仅 POE2** |
| **poe.ninja** | 通货汇率、物价统计 | POE1 + POE2 |
| **poeprices.info** | AI 智能估价 | POE1 + POE2 |
| **poewiki.net** | Wiki 百科查询 | POE1 + POE2 |
| **poedb.tw** | 数据库查询 | POE1 + POE2 |
| **Craft of Exile** | 装备配装模拟器导入 | POE1 + POE2 |
| **GitHub Releases** | 自动更新 | - |

---

> **说明**：本文档仅梳理 POE2 相关功能。POE1 独有功能（如财富追踪 Wealth Tracker）已排除。部分功能在 POE2 国服环境下可能需要额外的适配处理。
